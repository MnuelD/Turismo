# Plataforma de Turismo de Angola

Dashboard administrativo para a plataforma de turismo de Angola.

## Funcionalidades
- Gestão de estabelecimentos turísticos
- Sistema de reservas online
- Avaliações e comentários
- Multi-idioma e multi-moeda
- Dashboard analítico
- Gestão financeira

## Tecnologias
- HTML5, CSS3, JavaScript
- Bootstrap 5
- ApexCharts para gráficos
- Netlify para deploy

## Como Executar Localmente
```bash
npm install
npm run dev
